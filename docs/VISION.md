# 약관 심사 도구 — 개선·활용 비전

> 작성 2026-07-12. Phase 1(타사 비교)·2(표준약관 준수성)·3(감독규정 연결)+랭킹정밀도 완성 시점의 로드맵.

## 현 자산

3계층 코퍼스를 한 SQLite에 담은 **조문 지식 그래프의 뼈대**:
- **타사 약관(TERMS)**: 10개사·조항 ~102만
- **표준약관(STANDARD)**: 생보·질병상해·실손5세대
- **감독규정·법령(REG)**: 보험업감독규정·보험업법·시행령·시행규칙

엔진: LLM 없는 결정론적 유사도(FTS→문자 n-gram 코사인+BM25+제목가중+부정어 페널티). 폐쇄망 반입 가능(런타임 stdlib).
UI: 검색 / 단건 유사 / 붙여넣기 / docx 일괄심사 / 3섹션(표준약관·감독규정·타사).

## 개선 우선순위

### ① 표준약관 브릿지 (감독규정 매핑 2/6 정공법)
초안→감독규정 직접 유사도는 어휘 차이로 약함(현 2/6). 표준약관은 감독규정 별표15에서 나온 것이라
**초안→표준약관(강한 유사도)→감독규정(구조적 근거)** 경로로 우회하면 직접 매핑보다 정확.
주제사전보다 유지보수 부담이 작음.

### ② 차이 하이라이트(diff) — 준수성 심사의 핵심 가치
현재는 초안·표준약관을 나란히 보여줄 뿐. 단어 단위 diff로 "표준약관엔 있는데 초안에 빠진/추가된 문구"를
즉시 식별. 심사자의 눈대조를 도구가 대신 → 대량 약관검토 효율화의 본질. 엔진 위 렌더링만 추가.

### ③ 명시 인용 그래프 — 정밀도 100% 보조축
약관·규정 본문의 명시 인용("보험업법 제108조")을 파싱해 하이퍼링크. 커버리지 ~1%로 낮지만 정밀도 완벽 →
유사도(재현율 높음)와 상보적. 조문 열람 시 인용 법령 즉시 펼침.

### ④ 오탐·커버리지 정리
배당 랭킹 미스(정답 DB에 있으나 밀림), 실손 노이즈 4건, doc188(이미지 PDF) OCR. 신뢰도 직결.

## 활용 비전

- **약관검토 워크플로우 통합**: 상품부서 초안 → 자동 1차 심사 리포트(표준약관 diff+규정 근거+타사 관행) →
  법무 검토자는 이슈 플래그된 조문만 집중. 전수 검토 → 예외 집중으로 전환.
- **역방향 규제 대응**: 감독규정 개정 → 영향받는 자사 약관 조문 자동 역검색. impact 분석 자동화.
- **판례·유권해석 연결**: J-ID 판단자산·korean-law 판례를 조문에 연결 → 규범을 넘어 실제 분쟁 사례까지.
- **온보딩/교육**: 신입 심사자가 조문별 업계 관행·규범을 즉석 학습.

## 추천 순서

② diff 먼저(엔진 재사용·즉각 체감·before/after 명확한 자동화 사례) → ① 표준약관 브릿지(매핑 근본 해결) → ③④ 병행.
