# 표준약관 적재 (Phase 2) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 표준약관 3종(생명보험·질병상해·실손5세대)을 DB에 적재하고, 유사조문 결과 최상단에 "표준약관 대응 조문"을 항상 별도 강조해 준수성 심사를 지원한다.

**Architecture:** Phase 1 엔진(simmatch)·검색앱은 코어 불변. 표준약관을 로컬 vault에서 매니페스트 기반으로 적재(`doc_type='STANDARD'`), `db_similar`에 `doc_type` 필터를 더해 핸들러가 STANDARD/TERMS를 각각 조회한다.

**Tech Stack:** Python 3.14, SQLite(FTS5), PyMuPDF(fitz, 빌드전용), pytest. 런타임 엔진·검색앱은 표준 라이브러리만.

## Global Constraints

- **표준약관은 규범** — 각 소스의 개정일/세대를 `version_label`로 명시하고 적재 리포트에 정본 확인 결과를 남긴다.
- `db_similar`의 새 `doc_type` 파라미터 **기본값은 `'TERMS'`** — 기존 호출부(타사 검색) 동작 불변(하위호환).
- 표준약관엔 자사 개념 없음 → STANDARD 조회 시 `exclude_member` 미사용.
- 적재(ingest)는 빌드 시점(fitz 등 허용). **런타임 엔진(simmatch)·검색앱은 표준 라이브러리만**.
- STANDARD 발행처 member_cd: 생보=`STD_L`, 질병상해=`STD_H`, 실손=`STD_S`. 발행처명이 곧 UI 라벨.
- 테스트는 bare import(`import ...`), `tests/conftest.py`가 `src/`를 경로에 추가(Phase 1에서 구축).
- 경로 기준 `~/yakgwan-db`, 스크립트는 `src/`에서 `.venv/bin/python` 실행.
- 코드 스타일은 기존 `src/*.py`(한글 주석, 짧은 함수, 표준 라이브러리 우선).

Create `src/standards_prep.py`:
```python
#!/usr/bin/env python3
"""표준약관 소스(합본 md/초점 md/PDF) → 평문. 매니페스트 기반. (빌드 시점)

합본은 마크다운 표(| |) 래핑 + '######' 헤더를 씀 → 평문화 후 clause_split로 분할.
"""
import json
import pathlib
import re

ROOT = pathlib.Path(__file__).resolve().parent.parent
MANIFEST = ROOT / "catalog" / "standards.json"

_SEP_ROW = re.compile(r"^\s*\|[\s:|-]*\|\s*$")   # 표 구분행/빈 표행
_CELL = re.compile(r"^\s*\|(.*)\|\s*$")
_HDR = re.compile(r"(?m)^\s{0,3}#{1,6}\s+")


def strip_md_table(text: str) -> str:
    out = []
    for ln in text.split("\n"):
        s = ln.rstrip()
        if _SEP_ROW.match(s):
            continue
        m = _CELL.match(s)
        if m:
            s = m.group(1).strip()
        out.append(s)
    return "\n".join(out)


def strip_md_headers(text: str) -> str:
    return _HDR.sub("", text)


def slice_section(text: str, start_pat: str, end_pat):
    lines = text.split("\n")
    si = ei = None
    for i, ln in enumerate(lines):
        if si is None and re.search(start_pat, ln):
            si = i
        elif si is not None and end_pat and re.search(end_pat, ln):
            ei = i
            break
    if si is None:
        return text
    return "\n".join(lines[si:ei] if ei is not None else lines[si:])


def to_plaintext(entry: dict) -> str:
    p = pathlib.Path(entry["source_path"])
    if entry["source_type"] == "pdf":
        from extract_index import extract_pdf_text
        raw = extract_pdf_text(p)
    else:
        raw = p.read_text(encoding="utf-8")
        raw = strip_md_table(raw)
        raw = strip_md_headers(raw)
    if entry.get("section_start"):
        raw = slice_section(raw, entry["section_start"], entry.get("section_end"))
    return raw


def load_manifest():
    return json.loads(MANIFEST.read_text(encoding="utf-8"))
```

```python
#!/usr/bin/env python3
"""표준약관 평문 → DB 적재(doc_type=STANDARD). 재실행 안전(발행처별 기존 STANDARD 교체).

사용법: .venv/bin/python src/ingest_standards.py
"""
import hashlib

from clause_split import split_clauses
from common import open_db
from standards_prep import load_manifest, to_plaintext


def ingest(conn, entry: dict, plaintext: str) -> int:
    mcd, name = entry["member_cd"], entry["name"]
    conn.execute("INSERT OR REPLACE INTO insurers(member_cd, name) VALUES(?,?)", (mcd, name))
    # 발행처의 기존 STANDARD 문서·조문 제거(멱등)
    old = [r[0] for r in conn.execute(
        "SELECT doc_id FROM documents WHERE member_cd=? AND doc_type='STANDARD'", (mcd,))]
    if old:
        conn.executemany("DELETE FROM clauses WHERE doc_id=?", [(x,) for x in old])
        conn.executemany("DELETE FROM documents WHERE doc_id=?", [(x,) for x in old])
    sha = hashlib.sha256(plaintext.encode("utf-8")).hexdigest()
    cur = conn.execute(
        """INSERT INTO documents(member_cd, prod_nm_raw, doc_type, version_label,
               file_path, sha256, prod_group)
           VALUES(?,?,?,?,?,?,?)""",
        (mcd, name, "STANDARD", entry["version_label"], entry["source_path"], sha, entry["key"]))
    doc_id = cur.lastrowid
    chunks = split_clauses(plaintext)
    conn.executemany(
        "INSERT INTO clauses(doc_id, seq, clause_no, title, text) VALUES(?,?,?,?,?)",
        [(doc_id, i, no, ti, tx) for i, (no, ti, tx) in enumerate(chunks)])
    conn.commit()
    return sum(1 for no, _, _ in chunks if no and no.startswith("제"))


def main():
    conn = open_db()
    for e in load_manifest():
        txt = to_plaintext(e)
        n = ingest(conn, e, txt)
        print(f"  [{e['key']}] {e['name']} ({e['version_label']}) — 조문 {n}개 적재")
    total = conn.execute("SELECT COUNT(*) FROM documents WHERE doc_type='STANDARD'").fetchone()[0]
    print(f"완료: STANDARD 문서 {total}건")
    conn.close()


if __name__ == "__main__":
    main()
```
> `open_db()`는 `schema.sql`을 적용하며 `documents.doc_type`은 기존 TEXT 컬럼이라 STANDARD 값 사용에 스키마 변경 불필요. `sha256`은 평문 기반이라 소스 변경 시 자연 갱신.
